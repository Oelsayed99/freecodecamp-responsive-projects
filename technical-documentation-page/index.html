What is The Viewport?

The viewport is the user's visible area of a web page.

The viewport varies with the device and will be smaller on a mobile phone than on a computer screen.

Before tablets and mobile phones, web pages were designed only for computer screens.

Setting The Viewport

<meta name="viewport" content="width=device-width, initial-scale=1.0" />
What is a Grid View?

Many web pages are based on a grid-view, which means that the page is divided into rows and columns.

Building a Grid View

* { margin: 0; box-sizing: border-box; }
What is a Media Query?

Media query is a CSS technique introduced in CSS3.

Example

@media only screen and (max-width: 600px) { body { background-color: lightblue; } }
  • Adapts layout based on screen size
  • Makes sites mobile-friendly
  • Often uses breakpoints (in px, %, or em)
Add a Breakpoint

Media queries can change layout at certain screen widths (called breakpoints).

Example

@media only screen and (max-width: 600px) { .item1 {grid-area: 1 / span 6;} .item2 {grid-area: 2 / span 6;} }
Another Breakpoint

You can define multiple breakpoints for better flexibility.

Example

@media only screen and (min-width: 600px) { .item2 {grid-area: 2 / span 1;} } @media only screen and (min-width: 768px) { .item4 {grid-area: 2 / span 1;} }
